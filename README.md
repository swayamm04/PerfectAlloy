# PAC - Professional Admin Console

This repository contains the full-stack implementation of the Professional Admin Console.

## Project Structure

- **frontend/**: Next.js (App Router) administration dashboard.
- **backend/**: (In Progress) Backend services and API.

## Getting Started

### Frontend

To run the frontend dashboard:

1.  Navigate to the directory:
    ```sh
    cd frontend
    ```
2.  Install dependencies:
    ```sh
    npm install
    ```
3.  Start the development server:
    ```sh
    npm run dev
    ```
